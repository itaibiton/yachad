Create `src/shared/components/flights/flight-utils.ts` with these exports:

**1. `buildWhatsAppUrl(phone: string, flight: FlightData): string`**
- Strip non-digits from phone: `phone.replace(/\D/g, "")`.
- Look up departure and destination Hebrew names via `getCountryByCode()` from `@/shared/data/countries`.
- Format date in Hebrew locale: `new Date(flight.departureDate).toLocaleDateString("he-IL", { day: "numeric", month: "long", year: "numeric" })`.
- Build Hebrew message: `` `שלום, אני מעוניין/ת בטיסה מ${departure} ל${destination} בתאריך ${dateStr}. אנא ספרו לי עוד פרטים.` ``
- Return `https://wa.me/${cleanPhone}?text=${encodeURIComponent(message)}`.
- WhatsApp message is ALWAYS in Hebrew per user decision (users and agents are Israeli).

**2. `getUrgencyInfo(departureDate: number): { isUrgent: boolean; hoursLeft: number; minutesLeft: number }`**
- Calculate `diff = departureDate - Date.now()`.
- `isUrgent = diff > 0 && diff < 24 * 60 * 60 * 1000`.
- `hoursLeft = Math.floor(diff / (60 * 60 * 1000))`.
- `minutesLeft = Math.floor(diff / (60 * 1000))`.

**3. `getStatusVariant(status: string): "default" | "secondary" | "destructive"`**
- "available" -> "default"
- "full" -> "secondary"
- "cancelled" -> "destructive"

**4. `formatFlightPrice(price: number, currency: string): string`**
- Use `Intl.NumberFormat` with locale "he-IL" and style "currency" with the given currency code.
- Wrap in try-catch; fallback to `${price} ${currency}` if Intl fails on unknown currency code.

**5. Define `FlightWithAgent` TypeScript type** (exported):
- All fields from a flight document plus denormalized agent fields: `agentName: string`, `agentIsVerified: boolean`, `agentImageUrl?: string`.
- Use `Doc<"flights">` from convex if available, otherwise define manually matching schema fields plus `_id` and `_creationTime`.

Note: This file is NOT a "use client" file — it's pure utility functions with no hooks. Import the Convex `Doc` type from `convex/_generated/dataModel` if the generated types exist, otherwise define FlightWithAgent manually.
