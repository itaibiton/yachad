Implement the **"N new articles" snapshot banner** logic inside NewsGrid (Pattern 6 from RESEARCH.md):
- `useState` for `seenCount` (number | null) and `pendingCount` (number)
- On first load (status !== "LoadingFirstPage" && seenCount === null), set `seenCount = results.length`
- When `results.length > seenCount`, set `pendingCount = results.length - seenCount`
- Render `<NewArticlesBanner count={pendingCount} onShow={applyNewArticles} />` above the results when `pendingCount > 0`
- `applyNewArticles` sets `seenCount = results.length`, `pendingCount = 0`, and scrolls to top

Implement **pull-to-refresh on mobile** (locked CONTEXT.md decision — RESEARCH.md recommends 80px threshold touchmove listener triggering refetch):
- Add `useState` for `isPulling` (boolean) and `pullDistance` (number)
- Attach `onTouchStart` / `onTouchMove` / `onTouchEnd` handlers to the scrollable container wrapper div
- **`onTouchStart`:** Record `startY` via `useRef`. Only activate if `window.scrollY === 0` (user is at top of page)
- **`onTouchMove`:** If pull-to-refresh is active (started at scroll position 0), compute `deltaY = touch.clientY - startY`. Set `pullDistance = Math.min(deltaY, 120)`. If `deltaY > 80`, show the pull indicator (set `isPulling = true`)
- **`onTouchEnd`:** If `isPulling` is true and `pullDistance >= 80`:
  - Show a refreshing spinner/text at the top (use `t("refreshing")` from i18n)
  - Reset `seenCount` to null to re-trigger the baseline count capture (this effectively "refreshes" the view since `usePaginatedQuery` is already reactive — the pull-to-refresh signals the user that they've checked for new content)
  - After a brief `setTimeout(300)`, reset `isPulling = false` and `pullDistance = 0`
- If `onTouchEnd` fires with `pullDistance < 80`, cancel: reset `isPulling = false`, `pullDistance = 0`
- **Pull indicator UI:** When `pullDistance > 0`, render a `<div>` above the results with:
  - `style={{ height: pullDistance, overflow: 'hidden' }}` for smooth pull effect
  - Show `t("pullToRefresh")` when `pullDistance < 80`, `t("refreshing")` when `>= 80`
  - ArrowDown icon (lucide-react) that rotates 180deg when threshold reached: `className={cn("transition-transform", pullDistance >= 80 && "rotate-180")}`
- Wrap with `md:hidden` — pull-to-refresh is mobile only (desktop uses the "N new articles" banner)
