**Create `src/shared/components/AlertBanner.tsx`** — `"use client"`:

A cross-page alert banner that renders between TopBar and main content when active alerts exist. Uses real-time Convex subscription and sessionStorage dismiss pattern.

Implementation:

1. **Query:** `useQuery(api.modules.alerts.queries.listActiveAlerts)` — returns array of active alerts or `undefined` while loading
2. **Dismiss state:** Initialize `dismissedIds` as `useState<Set<string>>(new Set())`. Populate from sessionStorage in a `useEffect` (client-side only — avoids hydration mismatch):
   ```typescript
   useEffect(() => {
     const stored = sessionStorage.getItem("yachad-dismissed-alerts");
     if (stored) {
       try { setDismissedIds(new Set(JSON.parse(stored))); }
       catch { /* ignore corrupt storage */ }
     }
   }, []);
   ```
3. **Dismiss handler:** Adds alert `_id` to the Set and persists to sessionStorage:
   ```typescript
   const dismiss = (alertId: string) => {
     setDismissedIds((prev) => {
       const next = new Set(prev);
       next.add(alertId);
       sessionStorage.setItem("yachad-dismissed-alerts", JSON.stringify([...next]));
       return next;
     });
   };
   ```
4. **Filter visible:** `const visibleAlerts = (alerts ?? []).filter((a) => !dismissedIds.has(a._id));`
5. **Render null** when `visibleAlerts.length === 0`
6. **Layout:** Stack alerts vertically, most recent first (they come ordered from the query):
   ```
   <div className="flex flex-col">
     {visibleAlerts.map((alert) => (
       <div
         key={alert._id}
         className={cn(
           "flex items-center gap-3 px-4 py-2.5",
           alert.severity === "urgent"
             ? "bg-red-600 text-white"
             : "bg-amber-500 text-amber-950"
         )}
         role="alert"
       >
         <AlertTriangle className="size-5 shrink-0" aria-hidden />
         <div className="flex flex-col flex-1 min-w-0">
           <p className="font-semibold text-sm">{alert.title}</p>
           <p className={cn(
             "text-xs truncate",
             alert.severity === "urgent" ? "text-red-100" : "text-amber-800"
           )}>{alert.content}</p>
         </div>
         <button
           onClick={() => dismiss(alert._id)}
           aria-label={t("dismiss")}
           className={cn(
             "shrink-0 rounded-full p-1 transition-colors",
             alert.severity === "urgent"
               ? "hover:bg-red-700"
               : "hover:bg-amber-600"
           )}
         >
           <X className="size-4" />
         </button>
       </div>
     ))}
   </div>
   ```
7. Use `useTranslations("alerts")` for the dismiss label
8. Import `AlertTriangle`, `X` from `lucide-react`
9. Import `cn` from `@/lib/utils`
10. Import `api` from `"../../../convex/_generated/api"` (AlertBanner is in src/shared/components/, one level above news/)
